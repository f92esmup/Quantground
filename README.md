# Quantground

Este es un repositorio desarrollado con **Rust** con el objetivo de crear estrategias de trading y todas las herramientas necesarias para su desarrollo desde cero.

La intención principal de este proyecto es **aprender las bases de todo** el ecosistema de trading algorítmico, comprendiendo cada componente al construirlo uno mismo. El objetivo no es obtener beneficios económicos, sino el aprendizaje técnico y teórico profundo.
